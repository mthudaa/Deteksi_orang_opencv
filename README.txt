1. Download repository Machine Learning ini dalam format zip.
2. Ekstrak Folder dalam file zip "Deteksi_orang"
3. Buka IDE python anda lalu import folder "Face Recognition" sebagai projek baru anda.(saya menggunakan vscode)
4. download library opencv dengan menuliskan "pip install opencv-contrib-python" pada terminal anda.
5. atur directory intepreter anda menggunakan python 3.9 (terbaru) pada kode program berformat .py anda.
6. klik run untuk memulai program dan klik escape (Esc) untuk menghentikan program.
